# XenonStore
XenonStore is a Windows 8(.0) Metro application that aims to look similar as Windows Store from 2012, providing different apps to sideload on your Windows 8.x machine. Yes, it works on Windows 8.1 too.

# What was used to made it?
The only thing I used to create it was **Microsoft Visual Studio Express 2012 for Windows 8**, one of the software that can still compile Windows 8 (NOT Windows 8.1!) apps.

# How does it work?
When you open it, it automatically fetches the main JSON with applications and showes you the list of apps. You can click on them, so that app details page will open. You can read description, version and other stuff. When you click Install it downloads the .appx package to your Pictures folder.
